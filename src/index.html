2. Securing your wallet

If someone finds your Recovery Phrase above then they will be able to access all of your coins. You can optionally create a Passphrase to prevent unauthorized access in the case someone finds a copy of your 24 words.

If you enter a password here then you will also need that password to access your wallet in the future. Do not forget or lose this password.

  • a Public Key that will be used to send your Particl tokens to

Offline Usage

You can use this tool without having to be online.

  1. In your browser, select File → Save As… and save this page on your computer.
  2. Double-click that file to open it in a browser on any (even offline) computer.
  3. Alternatively, download the file from the repository: https://github.com/particl/bip39-wallet/
